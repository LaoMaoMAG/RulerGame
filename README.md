### 创意工坊

"创意工坊" 允许玩家加载" 数据包" 更改游戏内容，玩家也可以选择自定义自己的 "数据包" 修改出专属的内容创作。其内置 "数据包商店" 可以一键安装想要加载的游戏内容，不过由于部分平台的规则限制，并不是所有版本都支持创意工坊的全部内容。

游戏本体本质其实也是 "数据包" ，游戏的客户端程序提供了一个类似加载器的框架，并不包含游戏内的如剧情之类的内容，其内容是依赖 "数据包" 文件所提供。

"数据包商店" 是一个可以在游戏内下载安装 "数据包" 的功能，游戏默认会内置一个官方下载源，个人可以按照开源的商店服务器代码搭建自己的下载商店源，可以在 "创意工坊" 的设置里添加第三方商店源。

### 版本差异

> 因部分平台规则限制，并不是所有版本都能享有官方发行的游戏的全部功能。

|  版本   | 差异  |
|  ----  | ----  |
| 官方版  | 完全的功能支持 |
| TapTap | 不支持数据包编辑器 / 只允许商店剧情包 / 不支持自定义商店源 |
| 好游快报  | 不支持数据包编辑器 / 只允许商店剧情包 / 不支持自定义商店源 |
